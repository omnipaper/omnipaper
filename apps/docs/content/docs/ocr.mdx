OCR makes scans and PDFs searchable. Set it up in **Settings → OCR** after first login.

Only **Mistral OCR** is available today.

## Setup

1. Create an API key in the [Mistral console](https://console.mistral.ai) under **API Keys**.
2. In **Settings → OCR**, pick **Mistral OCR**, paste the key, **Test connection**, **Save**.

Mistral bills per page. Monitor usage in your Mistral account.

## Supported files

OCR runs on upload for **PDFs** and **images** (JPEG, PNG, WebP). `.txt` and `.docx` are searchable without OCR.

Without OCR configured, PDF and image uploads still work but stay in **failed** status until you set up the provider and re run.
